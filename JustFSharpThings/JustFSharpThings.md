# Ten Things I Like about the F# Language
1. **Table Stakes**
  - .NET language (compiles to IL)
  - ML family (OCaml, Haskell, Rust, etc.)  
  - Functional (First class functions, immutability, etc.)
  - Object-oriented 
  - Indentation based syntax (No braces, no semicolons)
  - Linear compiler
  - Partial application 
   ```fsharp
    let add x y = x + y
    let f = add 2
    f 3 // 5
   ```
